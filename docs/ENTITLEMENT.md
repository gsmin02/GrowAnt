# 요금제 Entitlement (확정)

| 항목 | Free | Standard | Premium |
|---|---|---|---|
| 가격(월) | 0원 | 9,900원 | 29,900원 |
| 광고 | 있음 | 제거 | 제거 |
| AI 모델 | Flash | Flash | Flash / Pro 선택 |
| 사용량(내부) | 횟수 3회(토큰 무관) | 토큰제 — "더 많은 사용량" | 토큰제 — "가장 많은 사용량" |
| 피드백 수준 | 일반(Flash) | 일반(Flash) | 상세(Pro 선택 시) |

## 규칙
- 사용량은 내부적으로 **토큰제**로 관리하되, **UI엔 토큰 숫자를 노출하지 않고 퍼센트/정성 표현**으로 표시("남은 사용량", "더 많은 / 가장 많은 사용량").
- Free만 토큰과 무관한 **실행 횟수 3회**(추후 변동 가능).
- Premium은 '무제한'이 아니라 **'가장 많은 사용량'**.
- 모델 **선택 구조는 두되 현재 벤더는 Gemini 단일**. 추후 Claude/GPT 추가 시 사용량+요금제로 구분.
- 결제는 **mock**: 요금제 선택 시 tier 즉시 적용되어 기능 차이를 비교 체험(실 PG 미연동).

## 열린 항목
- 토큰 → 퍼센트 표시 매핑 기준 (미정)
